**Description:** Developed a full-stack database application designed to streamline asset tracking and management while prioritizing data security and integrity.

**Tech Stack:** PHP, SQL, JavaScript, HTML/CSS, MVC Architecture, MariaDB.

**Secure Architecture:** Implemented a Model-View-Controller (MVC) design pattern to separate business logic from the user interface, enhancing system maintainability and scalability.

**Cybersecurity Implementation:** Mitigated critical vulnerabilities—including SQL Injection and Cross-Site Scripting (XSS)—by utilizing prepared statements and robust input validation protocols.

**Risk Mitigation:** Engineered secure account access and session management features to protect sensitive organizational data and prevent unauthorized breaches.
